# GA Admin 产品体验审视与优化清单（2026-07）

本轮从“用户是否能理解当前状态、能否安全恢复、手机与键盘是否可完成核心操作”三个维度审视现有功能。以下 24 项均纳入本次实现与验证范围。

## 全局反馈与可访问性

1. **反馈语义自动识别**：成功、错误、普通提示使用不同视觉和无障碍语义，降低用户误判。
2. **错误使用即时播报**：错误反馈使用 `role=alert` / assertive live region，读屏用户能及时获知失败。
3. **普通状态温和播报**：成功和进行中状态使用 polite live region，避免打断当前操作。
4. **全局消息可关闭**：侧栏消息提供明确的关闭按钮，避免历史提示长期占位。
5. **反馈详情可复制**：长错误、升级诊断等可一键复制，便于反馈或排障。
6. **反馈支持重试动作**：共享反馈组件提供标准重试入口，供失败流程复用。
7. **全局忙碌状态可感知**：应用根节点暴露 `aria-busy`，辅助技术可识别长操作。
8. **统一键盘焦点样式**：按钮、链接、表单和自定义可聚焦元素均有清晰 `focus-visible`。
9. **减少动画偏好生效**：系统开启 reduced motion 后关闭弹层、骨架和过渡动画。
10. **手机触控目标增大**：小屏核心按钮/输入目标不低于 44px，降低误触。
11. **深色模式反馈对比度**：信息、成功、警告、错误反馈在深色背景下使用独立高对比配色。

## 模型选择器（对话、Goal、反思服务共用）

12. **模型搜索**：可按服务商名称或模型 ID 即时筛选。
13. **服务商/模型数量摘要**：打开后显示当前可选服务商和模型总数。
14. **无结果提供下一步**：区分“搜索无匹配”和“尚未配置模型”，给出可执行说明。
15. **上下方向键导航**：服务商列、模型列均可循环移动焦点。
16. **左右方向键跨列**：从服务商进入模型、从模型返回当前服务商。
17. **Home/End 快速定位**：可直接跳到当前列首项/末项。
18. **Escape 关闭弹层**：桌面和手机均支持 Escape 退出。
19. **关闭后恢复焦点**：弹层关闭后焦点回到原模型按钮，键盘流程不断裂。
20. **移动端锁定背景滚动**：手机模型选择期间页面背景不再跟随滑动。
21. **完整 ARIA 选择语义**：触发器、dialog、listbox、option、selected 状态完整标记。
22. **禁用原因可说明**：共享选择器支持把“运行中/切换中”等禁用原因写入标题和无障碍名称。

## 文件编辑安全与可发现性

23. **未保存内容防丢失**：关闭/刷新页面及切换到另一个文件前提醒；文件列表显示项目数，搜索显示结果数并支持一键清空。

## 升级恢复

24. **断电/重启升级恢复**：后端检测状态文件中的旧 PID，把未完成任务标记为“已中断”而非永久运行；前端显示中断原因、保留原进度、提供“重新开始升级”和复制诊断入口。

## 验证映射

- `web/src/lib/ux.test.mjs`：反馈语义、模型过滤/计数、升级恢复展示、文件切换保护。
- `web/src/App.smoke.test.jsx`：模型搜索、键盘跨列、Escape、焦点恢复、移动端滚动锁、反馈关闭、文件防丢失与搜索清理。
- `internal/version/version_test.go`：旧 PID 下载任务被识别为中断；当前进程内的正常下载仍保持运行。
- `web/src/lib/style.test.mjs` 与构建检查：视觉契约、CSS 和生产打包回归。
